# 제주시테니스대회 운영 가이드

---

## 📋 Part 1 — 단체전 운영 플로우

### 1-1. 사전 준비 (운영자 대시보드)

> 메뉴: `/dashboard` → 단체전 섹션

1. **대회 설정** (`/dashboard/settings`)
   - `event_type` = `team` 또는 `both`
   - `team_format`: `full_league`(풀리그) / `group_tournament`(조별+토너먼트) / `prelim_tournament`(예선+토너먼트)
   - `team_match_type`: `3_doubles` (3복식·2승 선승) / `5_doubles` (5복식·3승 선승)
   - `team_sets_per_rubber`: 러버당 세트 수 (1 또는 3)
   - `lineup_mode`: `captain_pin`(캡틴 PIN) / `admin_only`(운영자만)
   - `allow_player_reuse`: 선수 재사용 허용 여부

2. **부서(Division) 생성** (`/dashboard/divisions`)
   - 예: A부, B부, 시니어부 등

3. **클럽 & 선수 등록** (`/dashboard/teams/clubs`)
   - 클럽 추가 (캡틴 이름, **6자리 PIN**, 시드번호)
   - 각 클럽에 선수 명단 (개별 또는 CSV 일괄)
   - **주장 지정** → 해당 PIN으로 라인업 제출

### 1-2. 조편성 / 풀리그 생성

> 메뉴: `/dashboard/teams/groups`

- **풀리그 모드**: `🎯 풀리그 생성` 클릭 → N팀 모두 서로 대전
- **조별 모드**: 조당 팀 수 선택 (2~6) → `조편성 생성`
  - 1팀 조 발생 시 자동 차단 (3팀+조당2팀 등)

### 1-3. 캡틴 라인업 제출

> URL: `/lineup/{tie_id}` — 운영자가 대전 관리에서 **🔗 라인업 URL 복사** 버튼으로 각 캡틴에게 전달

1. 캡틴이 URL 접속 → **6자리 PIN 입력**
2. 각 러버별 선수 2명 선택 (드롭다운)
3. **🔒 라인업 봉인 제출** → 상대팀 제출 대기
4. 양팀 제출 완료 → 자동으로 라인업 공개
5. 상대팀 라인업 확인 + 점수 입력 가능

> **PIN 5회 실패 → 10분 자동 잠금** (관리자 해제 가능)

### 1-4. 점수 입력 (4가지 경로)

| 경로 | 용도 | 위치 |
|---|---|---|
| **캡틴** | 라인업 공개 후 점수 입력 | `/lineup/{tie_id}` |
| **러버 PIN** | 코트별 6자리 PIN | `/pin/team` |
| **현장 관리** | 경기장 스태프 | `/venue` → PIN 로그인 |
| **운영자 (대시보드)** | 운영본부 | `/dashboard/teams/ties` |

### 1-5. 토너먼트 자동 진출

- 조별 완료 시 `rpc_fill_team_tournament_slots` 자동 실행
- 본선 TBD 슬롯에 각 조 순위별 팀이 자동 채워짐
- 우승팀 결정 시 다음 라운드 자동 승격

### 1-6. 순위 확인

> `/dashboard/teams/standings`

- 자동 계산: 승/패/러버득실/득실차 기준
- **동률 발생** → 노란 표시 → 클릭하면 수동 순위 지정 모달
- 부서별로 분리 표시 (다부서 풀리그 지원)

## 🏟 Part 2 — 현장 경기관리 사용법

> URL: `/venue` → 경기장 PIN 입력

### 2-1. 로그인
- 경기장 PIN 4자리 이상 입력
- 30분 세션 (자동 만료)

### 2-2. 메인 화면 구성

```
┌─ 헤더: 경기장명 · 코트 목록 · 부서 필터 ────┐
├─ 배정 메시지                                  │
├─ ⏳ 배정 대기 N경기     [🎯 자동 배정] ───── │
│  [전체][예선][16강][8강][4강][결승] 탭       │
│  ┌─ 미배정 경기 + [코트 선택 ▼] [배정] ──┐ │
│  ...                                          │
├─ 각 코트 카드 (진행중 🔴 · 대기 🟡🟢 · 완료) │
│  - ▲▼ 순서 변경 버튼                         │
│  - 📍 다른 코트로 이동/해제 드롭다운         │
│  - ▶ 경기 시작 버튼 (첫 대기 경기)           │
│  - 점수 입력 버튼                             │
└───────────────────────────────────────────────┘
```

### 2-3. 코트 배정 (신규 기능)

**수동 배정:**
1. 배정 대기 영역에서 원하는 경기 찾기
2. **라운드 탭**으로 필터 (예선/16강/8강 등)
3. 드롭다운에서 코트 선택 → **[배정]** 클릭

**자동 배정:**
- **🎯 자동 배정** 버튼 → 현재 필터 적용된 경기를 부하 최소 코트에 자동 분배

**재배정 / 해제:**
- 코트 카드의 경기에 **📍** 버튼 → 드롭다운으로 다른 코트 선택하거나 **(해제)** 선택
- 진행 중 경기 이동은 confirm 경고

**순서 변경:**
- 대기 경기의 **▲▼** 버튼으로 같은 코트 내 순서 조정

### 2-4. 경기 시작

- 첫 번째 대기 경기에 **▶ 시작** 버튼 표시
- 클릭 시 상태 → `IN_PROGRESS`
- 단체전(tie)은 시작 버튼 없음 (캡틴/관리자가 라인업 제출 후 자동 진행)

### 2-5. 점수 입력 (개인전만)

- 경기 카드의 ✏ 버튼 → 모달 열림
- 점수 + 승자 입력 → 저장
- 단체전 점수는 여기서 **입력 불가** (캡틴/러버PIN/마스터PIN 경로 사용)

### 2-6. 부서 필터

헤더 하단의 부서 탭으로 내 담당 부서만 표시
- **전체 / A부 / B부** 등
- 미배정 경기 카운트는 현재 필터 기준

## 🛡️ Part 3 — 마스터 PIN (운영자 모드)

> URL: `/admin-pin` → 마스터 PIN 입력

### 3-1. 로그인
- `events.master_pin_hash`와 bcrypt 비교 (진짜 해시)
- 30분 세션
- 대회 설정에서 `rpc_set_master_pin`으로 변경 가능

### 3-2. 관리 페이지 (`/admin-pin/manage`)

**3개 탭:**

#### 🎾 개인전 탭
- **검색**: 팀명/경기번호/라운드로 검색
- **경기 잠금 해제**: 참가자가 PIN으로 확정한 경기를 다시 열기
  - 사유 입력 (선택)
  - `locked_by_participant = false`로 초기화
- **점수 수정**: 점수 입력 + 승자 선택 → 저장
- **조별 완료 시 자동 본선 채우기** (기존 로직)

#### 📋 단체전 탭
- **부서 탭** + **라운드별 그룹핑** (조별리그 / 16강 / 8강 / 4강 / 결승)
- **대전 선택 → 러버 목록 확장**
  - 라인업 확인 (✓ 공개 여부)
  - 러버별 점수 입력 / 수정
- **검증**:
  - 모든 세트 동점 차단
  - 3세트 스플릿 시 3세트 필수
  - 완료된 tie 재수정 시 `다음 라운드 데이터 손실 위험` 경고

#### 🔒 PIN 잠금 탭 (신규)
- 현재 잠긴 PIN 목록 (클럽/러버/로그인)
- 실패 횟수 + 남은 잠금 시간 표시
- **개별 해제** 또는 **전체 해제** 버튼
- PIN 5회 실패 → 10분 자동 잠금 → 관리자 해제 가능

### 3-3. 마스터 PIN 설정

> `/dashboard/settings` → 마스터 PIN 섹션

1. **신규 설정** 또는 **재설정**
2. PIN 입력 + 확인 (둘 다 동일)
3. `rpc_set_master_pin` 호출 → bcrypt 해시 저장

### 3-4. 주의사항

- **세션 30분**: 만료되면 재로그인 필요
- **PIN 실수 5회** → 10분 잠금 → 다른 관리자에게 해제 요청
- **점수 수정 이력**: `log_audit` 테이블에 자동 기록 (토큰 앞 8자리 + 대상 ID)
- **FK 안전성**: 이벤트 삭제 시 clubs/members FK가 `SET NULL`이라 러버/경기 기록은 유지

## 🔀 플로우 요약 (단체전 한 경기의 전체 여정)

```
[1] 운영자: 클럽 등록 + 캡틴 PIN 배부
      ↓
[2] 운영자: 조편성 또는 풀리그 생성 (ties 자동 생성)
      ↓
[3] 운영자: 코트 배정 (대시보드 or 현장관리)
      ↓
[4] 캡틴: /lineup/{tie_id} 접속 → PIN 인증 → 라인업 제출
      ↓
[5] 양팀 제출 완료 → 자동 reveal → 러버 생성
      ↓
[6] 현장 스태프: ▶ 경기 시작
      ↓
[7] 캡틴/러버PIN/현장: 러버별 점수 입력
      ↓
[8] 러버 과반수 달성 → tie 자동 완료 → 순위 재계산
      ↓
[9] 조별인 경우: 조 완료 → 본선 TBD 자동 채움
      ↓
[10] 토너먼트 진행: 결승까지 자동 승격
      ↓
[11] 우승 확정 → /dashboard/teams/standings 에서 최종 순위
```

## 🚨 문제 발생 시

| 증상 | 해결 |
|---|---|
| PIN 잠금 | 마스터 PIN으로 `🔒 PIN 잠금` 탭에서 해제 |
| 라인업 공개 안 됨 | `/dashboard/teams/ties` → 대전 선택 → **📋 라인업 공개** 수동 버튼 |
| 점수 수정 필요 | 마스터 PIN → 해당 탭 → 점수 입력 |
| TBD 슬롯 수동 채우기 | `/dashboard/teams/bracket` → 조별 완료 후 **✅ 슬롯 채우기** |
| 코트 이동 | 현장관리 📍 버튼 또는 대시보드 드래그 (PC만) |
